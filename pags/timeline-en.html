Timeline: from Babbage to Hollerith

Explore the historical sequence.

Difference Engine — Charles Babbage

Designed to automatically calculate and print mathematical tables. It was mechanical, with gears and wheels—a fundamental step towards the computer.

Semyon Korsakov — Card Storage

Used punched cards/registers to organize information, anticipating mechanical data indexing systems.

Keyboard Calculator — Torchi

Early examples of machines with keyboard input, making direct number entry easier.

Electromechanical Relay — Joseph Henry

Electromechanical components that enabled signal control and are precursors to logic circuits.

Analytical Engine — Charles Babbage

General-purpose computer concept: arithmetic unit, memory, control, and use of cards for instructions—designed but not fully built in its time.

Ada Lovelace — First Algorithm

Note on the analytical engine containing an algorithm to calculate Bernoulli numbers. Recognized as the first programmer.

Relational Machine — Alfred Smee

Proposed machines to represent and compare ideas mechanically, influential but rarely built.

Boolean Algebra and Symbolic Logic

Formalization of logic by George Boole and later advances that made it possible to treat logical operations algebraically—the basis for digital circuits.

Telettrophone — Antonio Meucci

A distance communication device using wires and electricity, considered a precursor to the modern telephone.

Single Column Calculator — Caroline Winter

Simple device created to facilitate arithmetic calculations in a single column, contributing to the evolution of mechanical calculators.

The Typewriter

Mechanical device that allowed faster and more efficient writing, revolutionizing written communication.

Jevons' Logical Plan

A symbolic logic system proposed by William Stanley Jevons, aiming to represent logical reasoning more clearly and structured.

Direct Multiplication — Edmund Barbour

A method proposed by Edmund Barbour to simplify multiplication, using a more direct approach.

Frege's Logic — Gottlob Frege

Frege published a book called *Begriffsschrift*. It was a work that showed how to write logic with symbols, creating ideas that became modern logic. This helped mathematics and philosophy become more precise.

NAND and NOR — Charles Pierce

Charles Pierce discovered how to emulate 16 logical operators using only 2, NAND and NOR, which became essential in binary systems.

Brazilian Calculator — Azevedo Coutinho

A never-finished calculator project, created by a Brazilian during the industrial revolution.

Tabulating Machine — Herman Hollerith

Used punched cards and electromechanical readers to speed up the US census processing—a milestone in data automation and the origin of the Tabulating Machine Company (future IBM).
