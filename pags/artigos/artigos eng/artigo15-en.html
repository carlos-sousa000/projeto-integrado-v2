Brazilian Calculator - Azevedo Coutinho

In 1884, Azevedo Coutinho, a Portuguese inventor living in Brazil, created a prototype of a mechanical calculator that added automatically. Unfortunately, the machine was never built.

Overview Coutinho envisioned a mechanical calculator capable of performing addition operations automatically, without manual intervention at each step of the calculation. The machine's purpose was to simplify and speed up numerical calculations, making them more precise and efficient.

Historical Context

In the late 19th century, various inventors sought to automate calculations. Coutinho's proposal was one of the first from Brazil, showing the influence of mechanization ideas that were already circulating in Europe.

How did it work?

The user would choose the place value (hundreds, tens, units, etc.) by moving a piece, then enter each number by pressing buttons that activated gears connected to a display. It had a button that allowed resetting and restarting calculations. Until the reset button was pressed, every new number clicked would be added to the account. For example, if you had entered the number 450 and clicked 50, the machine would show 500.
